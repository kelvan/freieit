Eingesetzte Software

Für die Umsetzung dieses Webprojekts wurde folgende Freie Software eingesetzt:

  • GNU Emacs und gedit für die Bearbeitung der Rohtexte und des Quellcodes
  • Inkscape für die Gestaltung der Layouts
  • Gimp für die Bearbeitung des Bildmaterials
  • Git und Mercurial für Versionsmanagement im Entwicklungsteam
  • Django für die Umsetzung der Webseite
  • Python als Programmiersprache
  • PostreSQL als Datenbanksystem

Als Betriebssysteme wurden von unseren Teammitgliedern verschiedene GNU-Linux Distributionen eingesetzt:

  • Debian
  • Lubuntu
  • Arch Linux
  • ... [bitte ergänzen]
